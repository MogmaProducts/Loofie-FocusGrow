# Loofie FocusGrow — はんだ付けガイド

## 使うもの

### キット同梱品

- PCB（右、左）
- マイコンはんだ付け用治具

### 別途用意するもの

- PCBソケット
- Seeed XIAO nRF52840
- はんだごて
- はんだ

## はんだ付け手順

⚠️はんだ付けは失敗時のリカバリーが難しいため、作業前に手順全体を確認しておくことをおすすめします。
⚠️はんだ付け作業の際はやけどや怪我などに気をつけて行ってください。

### 1. PCBソケットのはんだ付け

![](./images/sold001.png)
<sub>はんだ付けで使う部品</sub>

1. PCBにPCBソケットを置きます。PCBソケットは向きがあるので、画像のとおりに置いてください。

   ![](./images/sold002.png)

2. 最初は片側のみはんだ付けします。はんだ付けの際はPCBソケットが浮かないように注意してください。

   ![](./images/sold003.png)

3. 全てのPCBソケットの片側をはんだ付けします。右側のPCBはPCBソケットの向きに注意してください。

   ![](./images/sold004.png)
    <sub>右側のPCB</sub>
   ![](./images/sold005.png)
   <sub>左側のPCB</sub>

   > ⚠️ US / ISO それぞれで使用するPCBソケットの位置が異なります。全てのPCBソケットをはんだ付けしても問題ありませんが、干渉が気になる場合はどちらか一方のみはんだ付けしてもかまいません。
   >
   > ![](./images/sold006.png)
   > <sub>右側PCBのエンター付近</sub>

4. 片側のはんだ付けが終わったら、PCBソケットが浮いていないか確認します。複数箇所をはんだ付けしてしまうと修正が難しいため、片側だけはんだ付けした状態で浮きを確認するのがよいでしょう。

   ![](./images/sold007.png)
   <sub>PCBを横から確認する</sub>

5. 浮いていないことを確認したら、PCBソケットの反対側もはんだ付けします。

   ![](./images/sold008.png)
### 2. Seeed XIAO nRF52840 のはんだ付け

1. Seeed XIAO nRF52840(以後マイコン)に治具を装着し、左側のPCBに乗せる。治具はPCBとマイコンの位置を合わせるために利用する。

   ![](./images/sold009.png)
   <sub>浮いていないことを確認</sub>
   ![](./images/sold010.png)
   <sub>最初に端をはんだ付けするのでこの位置に治具を装着</sub>

2. 位置が決まったら一箇所をはんだ付けする。マイコンがPCBと密着しているかを確認する。

   ![](./images/sold011.png)

3. 四方をはんだ付けし、位置を決定します。マイコンはスルーホールと端面の両方にはんだを流してはんだ付けをするとより固定されます。

   ![](./images/sold012.png)
   ![](./images/sold013.png)

4. マイコンのスルーホール全てをはんだ付けします。連続してはんだ付けをするとマイコンが加熱するので注意してください。

   ![](./images/sold014.png)

5. 裏面のスルーホールも整えます。少量のはんだをつけると画像のように綺麗になります。

   ![](./images/sold015.png)

### 3. 動作確認

はんだ付け後、マイコン並びにPCBソケットの導通を確認します。

#### ファームウェアの書き込み

1. ファームウェアをダウンロードする。
  - [Releases](https://github.com/MogmaProducts/loofie-focusgrow-zmk-config/releases/tag/first-release)
2. マイコンとPCをUSBケーブルで接続する。
3. マイコンのリセットボタンを素早く2回押す
   ![](./images/sold016.png)
   <sub>先の細いものだと押しやすいです</sub>
4. 認識されたストレージにファームウェアを書き込む
  - Macの場合はドラッグ＆ドロップで書き込める

キーボードの動作確認

1. 左右のPCBにSHケーブルを接続する。SHケーブルは上下があるので画像を参照しながら向きに注意して差し込む。
   ![](./images/sold017.png)
   <sub>端子が基板側になるように差し込むのが正しい</sub>
   ![](./images/sold018.png)

2. 入力されるかテストします。USBケーブルをPCに接続し、動作確認を行ってください。
  - 現状としてZMK Studioにキースイッチテスターの機能がありません。
  - VIAのテスターなんかを使うとよいかと思います [VIA Tester](https://www.usevia.app/test)
