# Loofie FocusGrow

## Features

- **4.5° のナチュラルテンティング**により、手首への負担を軽減
- **Tadpole を採用したガスケットマウント**で、打鍵感と静音性を両立
- **アルミ製トッププレート × SLS Nylon ボトムケース**の組み合わせによる剛性と機能性
- **ロウスタッガード 4 行配列**、USエンター・ISOエンターに対応した**コンバーチブルデザイン**

スタビライザーは以下の種類に対応しています。
